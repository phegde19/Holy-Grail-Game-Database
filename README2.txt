README2 - Data Collection 
==============================================
CollectData.js
==============================================
This script uses Node.js and axios to collect the data from the public API and saves it to a local file named data.json

How to run:
==============================================
-Make sure Node.js is installed: https://nodejs.org

-Install Node dependencies in the terminal:
  -npm install or npm I

-Run the script:
  -npm run collect

Output:
==============================================
- data.json will be created or overwritten in the folder when it is run
- Contains all the fetched data in JSON format 

Dependencies:
==============================================
-Node.js
-axios [installed via npm]
